these are two apps mobile & web app

to run the web app

```bash

cd web

yarn

yarn dev

```

to run the mobile app

```bash

cd mobile

yarn

yarn start

```

P.S you need to the following domains to the allowed origins of the running backend for the auth to work inside allowedOrigins.js file.

```bash

'http://localhost:5173',
'http://localhost:8081'

```
